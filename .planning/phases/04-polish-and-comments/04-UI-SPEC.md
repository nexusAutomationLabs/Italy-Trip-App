## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, category badge padding, avatar overlap offset |
| sm | 8px | Inline element spacing, comment meta gap |
| md | 16px | Card internal padding, form field gap, section spacing |
| lg | 24px | Day header margin, panel section gap |
| xl | 32px | Between day rows, sidebar section gap |
| 2xl | 48px | Hero image height breakpoint anchor |
| 3xl | 64px | Page top/bottom padding on desktop |

Exceptions:
- Touch targets (RSVP button, tab bar items, three-dot menu): minimum 44px height — accessibility minimum
- Event card min-width: 260px for horizontal scroll layout
- Event card max-width: 320px to preserve readable line length
- Sidebar width: 240px fixed on desktop (lg+)
- Attendee avatar circles: 32px diameter; overflow +N badge at 24px diameter

## Interaction Contracts

### Horizontal card scroll (desktop)
- Each day row is a horizontally scrolling container (overflow-x: auto, scroll-snap-x: mandatory)
- Scroll arrows (ChevronLeft / ChevronRight, 32px touch target, muted background, terracotta on hover) appear at left/right edges when overflow is detected
- On mobile: no scroll arrows — native swipe only
- Cards snap to nearest card edge (scroll-snap-align: start, 16px scroll padding)

### Event card
- Min-width 260px, max-width 320px, height 160px (no cover photo) / 220px (with cover photo)
- Cover photo: full-width top image at 80px height, object-fit cover, rounded top corners
- No cover photo: top area replaced with category color strip (8px) at card top edge
- Hover state: box-shadow elevation, left border accent (2px solid primary)
- Click: opens EventDetailPanel (existing Sheet/Drawer pattern, unchanged)
- Category icon: 20px Lucide icon, bottom-right corner, muted-foreground color

Category icon assignments (Claude's discretion, locked here):
- Dinner: `Utensils`
- Excursion: `Camera`
- Group Activity: `Users`
- Travel: `MapPin`
- Open Day: `Sun`

### Day header
- Format: "DAY ONE" (small-caps label, 11px, semibold, letter-spacing 0.1em) above "Thursday, May 7" (20px semibold)
- Arrival day: left border 4px primary (terracotta), "Arrival Day" label in muted-foreground 12px below date
- Departure day: same left border treatment, "Departure Day" label
- Timezone notice: single line below itinerary page title — "All times are Italy time (CEST)" — 12px, muted-foreground, italic. Appears once at page top, not per-event.

### Sidebar (desktop, lg+)
- 240px fixed width, height: 100vh, sticky top-0
- Top section: app title ("Berwick goes to Tuscany 2026") in Playfair Display 16px italic, trip dates below in 12px muted-foreground
- Nav links: "Itinerary" (Map icon: `CalendarDays`) and "Map" (`Map` icon), 40px height, 16px horizontal padding, full-width
- Active link: terracotta left border 3px, primary text color, primary/10 background
- Bottom section: user avatar (32px circle) + display name, "Edit Profile" ghost button

### Mobile bottom tab bar
- Fixed bottom-0, full width, background: white/card, border-top
- Two tabs: "Itinerary" (`CalendarDays`) and "Map" (`Map`), 44px height each
- Active tab: primary (terracotta) icon + label color
- Inactive tab: muted-foreground icon + label color
- Safe area padding (env(safe-area-inset-bottom)) applied

### Profile modal
- Triggered from sidebar user section (desktop) or header user menu (mobile)
- Dialog with two fields: Display Name (text input) and Profile Photo (file upload, accept image/*)
- Photo upload: preview cropped to 64px circle above the upload button
- Save button: "Save Profile" (primary)
- Photo stored in Supabase Storage; avatar_url written to profiles table

### Event detail panel enhancements
- Hero image: if event has cover_image_url, show at top of panel, 160px height, object-fit cover, full width
- No hero: no placeholder — description starts at top of panel content
- Google Maps embed: 240px height iframe, full panel width, shown below description when address is present
- Address display: muted-foreground 13px text above the map embed
- Attendee list: horizontal avatar strip (AvatarStrip) below RSVP button, with names on hover tooltip

### Event form (address field)
- Address field replaces or augments location_url field
- Uses Google Places Autocomplete — standard text input with dropdown suggestions
- On selection: auto-fills address string, stores lat/lng in hidden fields
- location_url field (Google Maps link) remains separately editable

### Empty day state
- Replaces "Nothing scheduled yet" (current copy) with a prompt card:
  - Heading: "Nothing planned yet"
  - Body: "Add something for the group to do on this day."
  - Button: "Add Event" (ghost, full width of card, Plus icon)

### Photo upload (event cover)
- Optional field in event create/edit form
- Accept: image/jpeg, image/png, image/webp
- Max size: 5 MB (enforced client-side with error message before upload)
- Preview: 80px height thumbnail inline in form after selection
- Stored in Supabase Storage events-covers bucket, cover_image_url written to events table
